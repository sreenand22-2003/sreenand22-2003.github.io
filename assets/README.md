# 📁 assets/ — Static Files README

## What goes here:

### images/
- `profile.jpg` — Your profile photo (replace this!)
  - Recommended: 400×400px square
  - Format: JPG or PNG
  - If missing, the site shows animated initials "SV" as fallback

### sreenand_resume.pdf
- Your CV / Resume PDF
- To change the filename, update all references in `index.html`:
  ```
  Search: sreenand_resume.pdf
  Replace: your_new_filename.pdf
  ```

## Notes
- Keep images optimized (< 500KB) for fast loading on GitHub Pages
- Use tools like https://squoosh.app/ to compress images
